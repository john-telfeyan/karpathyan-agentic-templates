# Sub-agents

> Stub. Linked from Operating Rule **"Use sub-agents to protect context."** Notes
> on when and how to delegate to sub-agents.

## When to use

- Broad, multi-domain research or parallel work that would bloat the main context.
- Do **not** spawn one for a small, well-scoped job — the overhead isn't worth it.

## Patterns

- Fan out by domain, then synthesize the results yourself.
- Handoff format: **orient → tasks → acceptance criteria → gotchas.**
